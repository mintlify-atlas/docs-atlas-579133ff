HY-World 2.0 is an open-source multi-modal world model from Tencent that transforms text prompts, single images, multi-view photos, or casual videos into persistent, navigable 3D worlds. Unlike video-based world models, HY-World 2.0 produces real 3D assets — 3D Gaussian Splatting (3DGS) and meshes — that can be imported directly into Blender, Unity, Unreal Engine, and Isaac Sim.

## What HY-World 2.0 produces

HY-World 2.0 outputs real 3D assets in a single forward pass — not video frames. You get:

- **3D Gaussian Splatting** (`.ply`) — renderable in real time on consumer GPUs
- **Point clouds** (`.ply`) — colored, compressed 3D point maps in world coordinates
- **Depth maps** — per-view depth in camera frame (PNG visualization + NPY raw values)
- **Surface normals** — per-view normal maps (PNG)
- **Camera parameters** — extrinsics and intrinsics (`camera_params.json`)
- **COLMAP-format reconstruction** — for downstream SfM pipelines (optional)

## Get started in 3 steps

<Steps>
  <Step title="Install dependencies">
    Create a conda environment with Python 3.10, install PyTorch 2.4 with CUDA 12.4, and install project requirements.

    ```bash
    conda create -n hyworld2 python=3.10
    conda activate hyworld2
    pip install torch==2.4.0 torchvision==0.19.0 --index-url https://download.pytorch.org/whl/cu124
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Run reconstruction">
    Point WorldMirror 2.0 at a directory of images or a video file. Model weights download automatically from Hugging Face on first run.

    ```python
    from hyworld2.worldrecon.pipeline import WorldMirrorPipeline

    pipeline = WorldMirrorPipeline.from_pretrained('tencent/HY-World-2.0')
    result = pipeline('path/to/images')
    ```
  </Step>
  <Step title="Explore your 3D world">
    Find your outputs in `inference_output/<scene_name>/<timestamp>/` — open `gaussians.ply` in a 3DGS viewer or import it into your game engine of choice.
  </Step>
</Steps>

<Note>
  HY-World 2.0 currently has WorldMirror 2.0 (world reconstruction) available as open-source. Full world generation code (WorldNav, WorldStereo 2.0, HY-Pano 2.0) is coming soon.
</Note>
